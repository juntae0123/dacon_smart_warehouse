## 🚀 Status
- **Current Score:** 17.81 (RMSE)
- **Model:** Two-Stage Hurdle Model (LGBM)
- **Key Strategy:** Zero-Ratio 동적 조절 및 Upper-Tail Scaling

## 🔍 핵심 EDA 인사이트 (오늘의 성과)
1. **Zero-Ratio의 역설**: 훈련 데이터 분석 결과, 실제 지연이 0인 경우는 **2.73%**에 불과함. 억지로 0을 맞추는 방어보다 97%의 미세 지연을 인정하는 것이 점수 향상에 결정적이었음.
2. **715분의 실체**: 최대 지연 시간이 700분을 상회하는 극단적인 **Long-tail 분포**. 이 피크를 과소평가하면 RMSE 페널티가 기하급수적으로 커짐.
3. **병목 임계점**: `주문량 / (로봇 수 * 효율)` 지표가 1.0에 도달할 때 지연이 폭증하는 물리적 인과관계 확인.

## 🛠️ 모델링 전략
- **Hurdle Model**: 지연 여부(분류)와 지연 시간(회귀)을 분리하여 각각 최적화.
- **Extreme Value Calibration**: 모델이 보수적으로 예측하는 경향을 깨기 위해 상위 1% 예측값에 **약 5배의 가중치**를 부여하여 고지연 구간 저격.
- **Log Transform**: 타겟값에 `log1p`를 적용하여 큰 오차에 대한 모델의 민감도를 조절.
